How It Works

The agent follows the ReAct framework — Thought, Action, Observation — inside a sandboxed Docker container with no network access.

┌──────────────────────────┐ │ python -m agent.main │ └────────────┬─────────────┘ │ ┌────────────────────────┼────────────────────────┐ │ │ │ ▼ ▼ ▼ Default Mode --export-sft --evaluate (fix bugs) (export training data) (benchmark agent) │ │ │ ▼ ▼ ▼ react_loop.py sft_exporter.py evaluator.py │ │ │ ┌─────────┴─────────┐ │ ┌────────┴────────┐ │ │ │ │ │ ▼ ▼ ▼ ▼ ▼ llm_client.py guardrails.py sft_dataset.jsonl reset_sandbox eval_report.json (Gemini API) (security) (HuggingFace) (Docker) (scores) │ │ ▼ ▼ ┌────────────────────────────────────────┐ │ DOCKER CONTAINER (no network access) │ │ │ │ test_runner.py → runs pytest │ │ file_reader.py → reads source code │ │ file_writer.py → writes fixed code │ │ git_manager.py → git add, commit │ └────────────────────────────────────────┘

Try It Yourself

Everything runs on free tools. No credit card needed.

Clone the repo
git clone https://github.com/NakuSurrey/The-Autonomous-DevOps-Auto-Fix-Agent.git
Get a free Gemini API key
Go to aistudio.google.com/app/apikey — takes 30 seconds, no credit card.
Set up environment
cp .env.example .env # paste your API key in .env
Install & run
make setup && make demo
Or without Make: pip install -r requirements.txt && docker-compose up -d --build && python -m agent.main
